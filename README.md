# Cálculo mensal de consumo de energia elétrica ⚡
Este projeto em Python foi criado para praticar o ciclo **Entrada → Processamento →Saída, juntamento com operações matemáticas e o uso de números inteiros e decimais**.
O programa solicita o **nome do aparelho**, a **potência do aparaelho em watts**, o **tempo médio de uso diário do aparelho em horas** e mostra o consumo médio mensal em **kWh/mês** e o valor a ser pago por esse consumo em **R$**. Nos cálculos são utilizadas as seguintes fórmulas **(Potência *Horas* 30)/1000** e **ConsumoMensal * 0.75**, para calcular a energia consumida pelo aparelho no mês e o valor a ser pago por esse consumo respectivamente.
